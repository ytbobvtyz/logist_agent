✅ Проект: умный ассистент логиста.
Цель: создать агента, который с помощью MCP инструментов получает необходимые данные по yandex api и составляет оптимальный маршрут между заданными пользователем населёнными пунктами.

Интеграции:
API ключ от OPENROUTER находится в файле .env в корне проекта, переменная OPENROUTER_API_KEY. Используем модель openrouter\free
API ключ от yandex находится в файле .env в корне проекта, переменная YANDEX_MAPS_API_KEY.

Стек: 
streamlit
mcp
httpx
python-dotenv
openai

## Обработка ошибок Яндекс API
Если YANDEX_MAPS_API_KEY отсутствует или невалиден:
- Вернуть сообщение: "❌ API ключ Яндекс.Карт не настроен"
- Предложить использовать расчёт по прямой (формула гаверсинусов)

Если город не найден в геокодере:
- Вернуть: {"error": "Город 'X' не найден", "city": "X"}
- Агент должен сообщить пользователю и остановить пайплайн

## Streamlit UI (app.py)

- Заголовок: "🗺️ Твой логист-ассистент"
- Чат-интерфейс (как в ChatGPT)
- Поле ввода текста (st.chat_input)
- Боковая панель содержит:
    * статус подключения к MCP
    * Отображение истории сообщений (st.chat_message)
    * Выбор модели из openrouter (достаточно 3 платных и 3 бесплатных)
    * Окошко отладки, позволяющее просмотреть факт и успешность обращений к MCP
- Авто-скролл к последнему сообщению

   → Ожидаемый результат: оптимальный порядок городов
