# Запуск расширенной версии Logsit Agent

## Новые возможности

### ✅ Реализованные функции:
1. **Множество диалогов** - создавайте и переключайтесь между разными задачами
2. **Автосуммаризация** - каждые 10 сообщений создается краткая сводка
3. **Task State (память)** - система отслеживает цель диалога и уточнения

## Быстрый старт

### 1. Установка зависимостей
```bash

# Убедитесь, что установлены все зависимости
```

## Основные отличия от стандартной версии

### Интерфейс:
- **Левая панель:** Управление диалогами
- **Создание диалогов:** Кнопка "➕ Новый диалог"
- **Переключение:** Dropdown со списком всех диалогов
- **Информация:** Отображение статистики и сводок

### Функции:
- **Автосуммаризация:** После 10 сообщений пользователя
- **Память задачи:** Автоматическое отслеживание контекста
- **Сохрание:** Все диалоги сохраняются в БД `conversations.db`

## Управление диалогами

### Создание диалога:
1. Нажмите "➕ Новый диалог"
2. Введите название (или оставьте автоматическое)
3. Начните общение в новом контексте

### Переключение между диалогами:
1. Выберите диалог из dropdown списка
2. Нажмите "↻ Переключиться"
3. История чата обновится для выбранного диалога

### Удаление диалога:
1. Выберите диалог из списка
2. Нажмите "❌ Удалить диалог"
3. **Внимание:** Удаление безвозвратное!

## Решение проблем

### Ошибка "Агент не инициализирован":
1. Проверьте наличие `.env` файла с OPENROUTER_API_KEY
2. Убедитесь, что интернет-соединение работает
3. Перезапустите приложение

### Медленная работа:
1. Проверьте размер БД `conversations.db`
2. При большом количестве диалогов рассмотрите архивацию старых
3. Используйте более мощную модель LLM для суммаризации

### Не работает суммаризация:
1. Проверьте счетчик сообщений пользователя (должно быть ≥10)
2. Убедитесь, что API ключ OpenRouter валиден
3. Проверьте логи на ошибки LLM вызовов
