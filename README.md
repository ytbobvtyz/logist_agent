# Logsit Agent 🗺️

Умный логистический ассистент с поддержкой множества диалогов, автосуммаризацией и памятью задачи.

## 🚀 Быстрый старт

### Установка зависимостей

```bash
# Убедитесь, что установлены все зависимости
pip install -r requirements.txt
```

### Настройка API ключей

Создайте файл `.env` в корне проекта:

```env
# Обязательные ключи
OPENROUTER_API_KEY=sk-or-v1-ваш_ключ_openrouter

# Опциональные ключи
YANDEX_MAPS_API_KEY=ваш_ключ_yandex
```

### Запуск приложения

```bash
python app/main.py
```

Приложение будет доступно по адресу: http://localhost:7872

## ✨ Возможности

### 📋 Основные функции
- **Множество диалогов** - создавайте и переключайтесь между разными задачами
- **Автосуммаризация** - каждые 10 сообщений создается краткая сводка
- **Память задачи** - система отслеживает цель диалога и уточнения
- **Интеграция MCP** - расчеты маршрутов и стоимости доставки
- **RAG поиск** - поиск информации в документах перевозчиков

### 🛠️ Технологии
- **LLM**: OpenRouter API с поддержкой различных моделей
- **MCP**: Model Context Protocol для инструментов расчета
- **RAG**: Векторный поиск по документам (FAISS + Sentence Transformers)
- **UI**: Gradio с современным интерфейсом
- **База данных**: SQLite для хранения диалогов и сообщений

## 🎯 Использование

### Интерфейс
- **Левая панель**: Управление диалогами и настройки
- **Правая панель**: Чат с ассистентом
- **Создание диалогов**: Кнопка "➕ Новый диалог"
- **Переключение**: Dropdown со списком всех диалогов

### Примеры запросов
1. **Расчет маршрута**: "Найди оптимальный маршрут между Москвой, Санкт-Петербургом и Казанью"
2. **Поиск в документах**: "Какие тарифы на доставку до 50 кг?"
3. **Уточнение задачи**: "Добавь срок доставки 2 дня и вес 100 кг"

## 🔧 Конфигурация

Настройки приложения в файле `.env`:

```env
# API ключи
OPENROUTER_API_KEY=ваш_ключ
YANDEX_MAPS_API_KEY=ваш_ключ_yandex

# Настройки приложения
APP_HOST=0.0.0.0
APP_PORT=7872
DEBUG=false

# База данных
DATABASE_URL=sqlite:///conversations.db

# MCP серверы
MCP_SERVERS={"route_planner": "python route_planner/mcp_server.py"}
```
