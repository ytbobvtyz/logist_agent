# Умный ассистент логиста

Агент для планирования оптимальных маршрутов между городами с использованием MCP инструментов и Яндекс API.

## Возможности

- Геокодирование городов через Яндекс API
- Расчёт оптимального маршрута (задача коммивояжёра)
- Форматирование результатов в читаемый вид
- Gradio веб-интерфейс с чатом
- Поддержка бесплатных и платных моделей OpenRouter

## Структура проекта

```
route_planner/
├── mcp_server.py       # MCP сервер с 3 инструментами
├── agent.py            # LLM агент для вызова пайплайна
├── app.py              # Gradio UI
requirements.txt        # Зависимости
.env                    # API ключи (не в репозитории)
```

## Установка

1. Клонировать репозиторий
2. Создать виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```
3. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Создать файл `.env` в корне проекта:
   ```
   OPENROUTER_API_KEY=ваш_ключ_openrouter
   YANDEX_MAPS_API_KEY=ваш_ключ_yandex
   ```

## Запуск

Смотрите инструкции в файле `run.md`.

## Использование

### Веб-интерфейс
1. Запустите приложение: `cd route_planner && python app.py`
2. Откройте http://localhost:7862 в браузере
3. Введите запрос, например:
   - "Найди маршрут между Москвой и Санкт-Петербургом"
   - "Построй оптимальный маршрут: Москва, Казань, Нижний Новгород"

### Командная строка
```bash
# Стандартный тест
python quick_test.py

# Тест конкретного запроса
python quick_test.py --message "Найди маршрут между Москвой и Санкт-Петербургом"

# Интерактивный режим
python quick_test.py --interactive
```

Агент автоматически:
- Получит координаты городов
- Рассчитает оптимальный порядок
- Вернёт отформатированный маршрут

## MCP инструменты

1. **geocode_batch** — получает координаты городов через Яндекс API
2. **find_optimal_route** — находит оптимальный порядок обхода точек
3. **format_route_summary** — форматирует результат в читаемый вид

## Модели

### Бесплатные

- `stepfun/step-3.5-flash:free`
- `openrouter/free`

### Платные
- `xiaomi/mimo-v2-pro`
- `minimax/minimax-m2.5`
- `deepseek/deepseek-v3.2`

## Обработка ошибок

- Если API ключ Яндекс не настроен → используется расчёт по формуле гаверсинусов
- Если город не найден → возвращается сообщение об ошибке
- Если MCP недоступен → ответ даётся без использования инструментов
