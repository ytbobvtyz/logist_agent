# Запуск приложения

## Предварительные требования

- Python 3.10+
- API ключи OpenRouter и Яндекс.Карт

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Настройка API ключей

Создайте файл `.env` в корне проекта:

```env
OPENROUTER_API_KEY=sk-or-v1-ваш_ключ_openrouter
YANDEX_MAPS_API_KEY=ваш_ключ_yandex
```

## Запуск приложения

### Способ 1: Текстовый интерфейс (рекомендуется)

```bash
cd route_planner
python text_interface.py
```

### Способ 2: Командная строка

```bash
# Быстрый тест
python quick_test.py

# Тест конкретного запроса
python quick_test.py --message "Найди маршрут между Москвой и Санкт-Петербургом"

# Интерактивный режим
python quick_test.py --interactive
```

### Способ 3: Веб-интерфейс (экспериментальный)

```bash
cd route_planner
python app.py
```

Приложение будет доступно по адресу: http://localhost:7862

### Способ 2: Только MCP сервер (для тестирования)

```bash
cd route_planner
python mcp_server.py
```

MCP сервер запустится на стандартном вводе/выводе (stdio).

### Способ 3: Тестирование агента без UI

```bash
cd route_planner
python -c "
import asyncio
from agent import RoutePlannerAgent

async def test():
    agent = RoutePlannerAgent()
    await agent.connect_mcp()
    response = await agent.process_message('Найди маршрут между Москвой и Санкт-Петербургом')
    print(response)
    await agent.disconnect_mcp()

asyncio.run(test())
"
```

## Проверка работы

### Текстовый интерфейс (рекомендуется)
1. Запустите: `cd route_planner && python text_interface.py`
2. Выберите модель из списка
3. Введите тестовый запрос:
   ```
   Найди лучший маршрут между Москвой, Санкт-Петербургом и Казанью
   ```
4. Ожидаемый результат:
   ```
   🗺️ Оптимальный маршрут:
   
   Москва → Казань → Санкт-Петербург
   
   📏 Расстояния:
   • Москва → Казань: 820 км
   • Казань → Санкт-Петербург: 1500 км
   
   📊 Общее расстояние: 2320 км
   ```

### Командная строка
```bash
# Быстрая проверка
python quick_test.py

# Проверка конкретного запроса
python quick_test.py --message "Найди маршрут между Москвой и Санкт-Петербургом"

# Интерактивное тестирование
python quick_test.py --interactive
```

### Веб-интерфейс (экспериментальный)
1. Откройте http://localhost:7862 в браузере
2. Введите тестовый запрос

## Тестовые запросы

1. **Базовый маршрут:**
   ```
   Найди маршрут между Москвой и Санкт-Петербургом
   ```

2. **Несколько городов:**
   ```
   Как лучше объехать Москву, Казань, Нижний Новгород?
   ```

3. **Международные города:**
   ```
   Построй маршрут между Парижем и Берлином
   ```

4. **Ошибка: мало городов:**
   ```
   Маршрут
   ```
   Ожидаемый ответ: "Укажите хотя бы два города"

## Остановка

Нажмите `Ctrl+C` в терминале, где запущено приложение.

## Возможные проблемы

1. **Ошибка "API ключ Яндекс.Карт не настроен"**
   - Проверьте наличие файла `.env`
   - Убедитесь, что ключ указан верно

2. **Ошибка подключения к MCP**
   - Убедитесь, что все зависимости установлены
   - Проверьте, что Python версии 3.10+

3. **Модель не отвечает**
   - Проверьте баланс OpenRouter
   - Попробуйте бесплатную модель

4. **Порт 7860 занят**
   - Измените порт в `app.py` строке `server_port=7860`
